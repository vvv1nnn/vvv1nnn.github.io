What is the difference between Margin, Border, and Padding?

In a CSS box model, there are 4 major components: content, padding, border and margin.

  1. The content is the part of the box which has the text, images, icons, etc. An example of content is this text that is written here.
  2. Padding is the space between the content and the border. The white space between the text and the border in this box is an example of padding.
  3. The border can be either a visible or invisible line around the edge of the padding. In my webpage, the black border is visible around the padding and it also has rounding on the edges for an aesthetic look.
  4. THe margin is the space around the border; it is the space between the border and the edge of the screen. It can be used to create responsive webpages.
